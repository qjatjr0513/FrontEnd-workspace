이벤트 핸들러(이벤트 발생시 실행될 function) 연결 방법 종류

* 방법1 : 이벤트 메소드를 통한 연결

        $("선택자").이벤트 메소드(function(){
            해당 요소에 해당 이벤트 발생시 실행할 내용
        });

클릭해보세요 이거.



* 방법2 : on 메소드를 이용한 방법

        $("선택자").on("이벤트명", function(){
            해당 요소에 해당 이벤트 발생시 실행할 내용
        });

자 마우스를 올려서 클릭 해보세요.



* 방법3

        $("상위요소 선택자").on("이벤트명","하위요소 선택자",function(){
            선택된 상위요소 안에 존재하는 해당 하위요소에 해당 이벤트가 발생했을때
            실행할 내용
        });

h4입니다. 클릭해주세요

h5입니다. 클릭해주세요

* 단, 동적으로 만들어진 요소에도 동일한 이벤트를 적용시키고자 한다면 반드시 방법3로 진행.

* 일회성 이벤트

이벤트를 딱 한번만 연결할때 사용
$("선택자").one("이벤트명",function(){
이벤트가 발생시 실행할 내용
})

단 한번만 실행하자.

* 키보드 관련 이벤트

- keydown / keypress : 키가 눌려질때 발생하는 이벤트
- keydown : 키보드의 모든 키가 눌려질때 발생됨
- keypress : 키보드의 펑션키, 기능키, 한글 제외한 나머지 키가 눌려질때 발생

- keyup : 키가 떼어질때 발생하는 이벤트
