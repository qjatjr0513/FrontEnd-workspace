테두리 관련 스타일 (border~)

        선택자{
            * 테두리 스타일 
            border-위치-style : none/dotted/dashed/solid/double/groove/ridge/inset/outset

            * 테두리 두께
            border-위치-width : 테두리 두께;

            *테두리 색상
            border-위치-color : 테두리 색상;

            * 테두리 두께, 스타일, 색상 한번에 지정가능.
            border-위치 : 두께 스타일 색상; => 색상생략시 글자색과 통일.

            * 테두리 모서리를 둥그스름하게 표현
            border-위/아래 - 좌/우-radius: 숫자; -> 위치 생략시 사방이 둥구스룸해짐.

            * 박스에 그림자 효과
            box-shadow : 가로거리 세로거리 흐림정도 번짐정도 색상;
        }
        위치=> 위/아래/좌/우(top/bottom/left/right) : 생략가능.
