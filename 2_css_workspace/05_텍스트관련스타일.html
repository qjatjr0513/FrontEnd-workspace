텍스트 관련 스타일(text~)

color : 텍스트의 색상을 지정

        선택자{
            color : 색상명/16진수/rbg(x,x,x)/rgba(x,x,x,x)/..
        }

text-decoration : 텍스트에 줄을 긋거나 없앨때 사용

        선택자{
            text-decoration : none/underline/overline/line-through;
        }
  • underline : 텍스트 영역 아래에 줄긋기
  • line-through : 텍스트 영역 중간에 줄긋기
  • overline : 텍스트 영역 위에 줄긋기
네이버로

text-transform : 영문텍스트의 대소문자 변환시 사용.

        선택자{
            text-transform : uppercase/lowercase/captalize;
        }
  • uppercase : 모든 영문자 다 대문자로
  • lOweRcase : 다 ~소문자로
  • captalize hi hey : 영문자의 첫 글자만 대문자로.

text-align : 텍스트를 정렬할때 사용

        선택자{
            text-align: left(기본값)/justify/right/center;
        }

text-shadow: 텍스트에 그림자 효과를 줄때 사용

        선택자{
            text-shadow : 가로거리 새로거리 번짐정도 색상;
        }
        - 번짐정도 색상은 생략가능.
        - 가로거리와 세로거리가 양수일경우 => 오른쪽 아래 그림자 생성
        - 가로거리와 세로거리가 음수일 경우 => 왼쪽 위에 그림자 생성
        - 번짐 정도가 0일 경우 그림자 색이 제일 진함.
        - 색상은 기본값으로 글자색과 동일.
