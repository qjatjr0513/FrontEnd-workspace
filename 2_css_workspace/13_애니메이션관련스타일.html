애니메이션 관련 스타일

애니메이션

속성을 점차적으로 변화시켜 움직이는 효과를 내는 기법. 트랜지션과 비교했을때 시작스타일과 끝 스타일을 부드럽게 이어 주는 기능을 제공한.
애니메이션 중간에 원하는 위치에서 @keyframes을 이용하여 중간스타일 지정 가능.




keyframes

애니메이션 시작지점과 끝지점의 스타일을 정하고, key frames의 이름을 지정. 시작지점과 끝 지점 두개만 설정할 경우 from to로 설정. 중간에 작업이 더있으면 %로 구분하여 설정. 

            @keyframes 키프레임명{
                석택자 {스타일;}
            }
