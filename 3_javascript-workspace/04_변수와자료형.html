변수와 자료형

* 변수 선언위치에 따른 전역변수/지역변수 

        <script> 
            let 변수명; //전역변수(globall)
            const 변수명; // 상수(값을 할당 바꿀수없음.)

            var 변수명; // 전역변수(global) 과거에 자주사용되던 변수.

            변수명; // 전역변수

            function 함수명(){
                let 변수명; // 지역변수(local)

                변수명; //지역변수(local)
            }

            var변수는 문제점이 많아서 let, const 대체됨.
        </script>

* 자료형

변수 선언시 자료형을 별도로 지정하지 않음(자료형 개념이 없진 않음.)
=> 변수에 대입되는 값(리터럴)에 따라서 자료형이 자동으로 결정됨.

자바스크립트의 자료형

  • string (문자열)
  • number (숫자)
  • BingInt (엄청~큰 숫자)
  • boolean (논리값)
  • object (객체)
  • function (함수)
  • undefined (초기화가 안된 변수)
