문단을 나누는 태그

문단 영역을 나누는 태그로는 p태그와 pre태그가 있다.
단, p태그는 줄바꿈 입력을 별도의 태그로 지정해야한다.
그리고 공백은     한개의 공백만을 표시하기 때 문에 기호문구를 작성해야한다.  

    pre          태그는 여러 공백이 인식 가능하고
    줄바꿈 등을 포함하여 입력한 내용 그대로 표현하는 
    태그

그 밖에 글자를 다루는 태그들

일반글꼴(태그를 감싸지 않은 텍스트)

b : 글자를 굵게 표시하는 태그

strong : 글자를 굵게 표시하는 태그

em : 글자를 기울여서 표시하는 태그

i : 글자를 기울여서 표시하는 태그

mark: 형광펜효과를 주는 태그

글자밑에 밑줄이 그어지는 태그

s : 글자에 취소선을 넣어주는 태그

strike : 글자에 취소선 넣어주는 태그

small : 글자를 작게 표현해주는 태그

big : 글자를 크게 표현해주는 태그

IOT란 ? 각종 사물에 센서와 통신기능을 내장해 인터넷에 연결하는 기술이다.

sub : 기본글자에 아래첨자를 나타내는 태그.

sup : 윗첨자를 나타내는 태그이다.
